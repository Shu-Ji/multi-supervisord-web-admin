multi-supervisord-web-admin
===========================

Mulitple Supervisord Web Admin.


Usage
===========================

    $ pip install -r requiretments.txt
    $ cd src
    $ python models.py
    $ python app.py

Browse http://127.0.0.1:8090 with your Google Chrome.
